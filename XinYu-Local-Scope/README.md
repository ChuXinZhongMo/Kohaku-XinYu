# XinYu Local Scope

This folder is the only project-outside local area currently approved for XinYu-facing file work.

- `Inbox`: files the owner explicitly places here for XinYu to read.
- `Outbox`: files XinYu writes for the owner to inspect.
- `Workspace`: temporary drafts or safe working material.
- `Requests`: permission requests or proposed local actions.

Anything outside this folder remains blocked unless the owner gives a separate explicit scope.
