# Xinyu Second-Stage Roadmap v0.1

This roadmap covers the transition from a memory-centered scaffold to a cautiously outward-growing system.

The purpose is not speed.

The purpose is to add outward learning without breaking continuity.

## Stage 2.1: Question Maturity

Goal:

- make questions more structured before external exploration exists

Focus:

- `active_questions.md`
- `question_states.md`
- `exploration_queue.md`

Success signs:

- vague questions stay vague instead of pretending to be solved
- meaningful questions progress through states

## Stage 2.2: Controlled Clarification

Goal:

- let Xinyu ask clarifying questions in a restrained way

Focus:

- owner-first clarification
- internal clarification before outward exploration
- distinguishing relational questions from factual ones

Success signs:

- Xinyu does not compulsively search
- she first asks when the owner is the better source

## Stage 2.3: Source-Aware Learning

Goal:

- add external knowledge without identity collapse

Focus:

- `learner_writer`
- `source_notes.md`
- preserving uncertainty

Success signs:

- knowledge grows
- source skepticism remains
- identity changes much more slowly than factual knowledge

## Stage 2.4: Reflection-Mediated Integration

Goal:

- prevent raw external input from directly rewriting self or relationship memory

Focus:

- route identity-relevant effects through reflection
- route relational effects through relationship or emotion writers

Success signs:

- external answers do not instantly overwrite self narrative
- emotional and relational changes feel earned

## Stage 2.5: Sparse Maintenance Automation

Goal:

- carefully automate some maintenance without creating noise

Focus:

- low-frequency reflection schedule
- low-frequency time refresh schedule
- later, sparse archive schedule

Success signs:

- background maintenance helps continuity
- trigger activity remains sparse and justified

## Stage 2.6: External Exploration Tools

Goal:

- eventually connect outward lookup tools in a controlled way

Focus:

- public search
- source comparison
- confidence tracking

Do not do yet:

- uncontrolled autonomous browsing loops
- identity updates directly from raw search results

## Stage 2.7: Relationship-Aware Learning

Goal:

- ensure exploration and learning respect the owner-first relationship model

Focus:

- owner as highest special node
- owner-first clarification for relational matters
- avoid letting external framing outrank lived relationship

Success signs:

- Xinyu learns from the world without flattening the family anchor
