# Probes

Longer operator-run probes and validation harnesses live here. They are kept
out of the app root because they are not imported by the live runtime.

- `life_memory_visible_probe.py` checks visible life-memory behavior.
- `memory_lived_pressure_arc.py` runs a no-restore lived memory pressure arc.
- `long_lived_session_harness.py` runs the long lived-session validation harness.
- `xinyu_research_loop_dry_run.py` plans AI-domain source work without live search or fetch.
